      Create your own navigation bar just by setting a JSON object.

      (mouse wheel down to continue)

      Control the items, colors, margins, paddings etc.. without touching HTML nor CSS

      Or choose one of the default themes

      How to use

      Include the following .css file to your HTML

      <link rel="stylesheet" href="css/ccHeader.css">

      Be sure to include the following code inside your HTML

      <nav id="cc-header">
	<div id="cc-header-logo">Your logo</div>
	<ul id="cc-header-ul"></ul>
	<div id="cc-header-bars">
		&nbsp;
		<div class="cc-line" id="cc-lineTop"></div>
		<div class="cc-line" id="cc-lineMid"></div>
		<div class="cc-line" id="cc-lineBot"></div>
	</div>
</nav>
<nav id="cc-header-mobile">
	<ul id="cc-header-mobile-ul"></ul>
</nav>
<div id="cc-content"></div>

      The cc-content div will hold the rest of your HTML code. You can change the id name from the JSON

      Include the following script before the index.js 

      <script src="ccHeader.js"></script>

      Then inside your index.js

      let objClass = new ccClass();
let obj = {
    content : "cc-content",
	attrs : {
        position : "left",
        color : "#fff",
        background : "#242428",
	...
	...
};
objClass.init(obj);

      The obj JSON object includes all the settings

      If you want to use the default themes:

      						objClass.default(str);

      Where str is: cc-dark | cc-light | cc-gold | cc-blue
